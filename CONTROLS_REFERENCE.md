# ROGUELANCER - QUICK REFERENCE CARD

## ?? ESSENTIAL CONTROLS

### THROTTLE
```
W     - Accelerate Forward
S     - Stop / Reverse
TAB   - Afterburner (150 speed)
Z     - Cruise Mode (300 speed)
X     - Kill Engines (drift)
R     - Match Speed (50%)
```

### MANEUVER
```
? / I   - Pitch Up
? / K   - Pitch Down
? / J   - Yaw Left
? / L   - Yaw Right
A       - Yaw Left (alt)
D       - Yaw Right (alt)
Q       - Roll Left
E       - Roll Right
```

### STRAFE (Hold Shift)
```
Shift + A   - Strafe Left
Shift + D   - Strafe Right
Shift + W   - Strafe Up
Shift + S   - Strafe Down
```

## ?? FLIGHT MODES

| Mode | Key | Speed | Effect |
|------|-----|-------|--------|
| Normal | - | 80 | Standard flight |
| Afterburner | TAB | 150 | Fast, less agile |
| Cruise | Z | 300 | Auto-speed |
| Reverse | S (hold) | -40 | Backward thrust |
| Engines Kill | X | Drift | Momentum only |

## ?? HUD INDICATORS

**Throttle Bar (Right):**
- Blue/Cyan = Normal
- Red = Afterburner
- Yellow = Cruise
- Orange = Reverse
- Gray = Engines Killed

**Status Box (Top Right):**
- Shows current flight mode

## ?? QUICK TIPS

1. **Auto-bank**: Ship banks automatically when turning
2. **Strafe**: Hold Shift for lateral movement
3. **Quick stop**: Tap S repeatedly
4. **Drift turn**: Kill engines + momentum turn
5. **Combat**: Afterburner + strafe for dodging
